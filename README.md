# Monoclaw Monorepo

Multi-lang hub (Rust/Python/C++/JavaScript).

## Structure
- **Rust/**: Cargo crates
- **Python/**: Poetry pkgs
- **Cpp/**: CMake projects
- **JavaScript/**: Static/web tools (HTML, JS, CSS)
- **examples/**: Demos
- **shared/**: Protos/FFI

## Usage
`git checkout -b feat/xxx`
Build per-lang.
